Mancala Rust AI
Rustと言語のGUIライブラリ egui を使用して開発された、AI対戦機能付きのマンカラ（カラハ）ゲームです。
CLIベースのロジックから始まり、最終的に直感的なGUIと強力な思考エンジンを搭載しました。

🚀 特徴
本格的なマンカラ・ロジック:

「石をまく（Sowing）」基本ルールに加え、「横取り（Capture）」や「ボーナスターン」を完全実装。

強力なAIエンジン:

ミニマックス法（Minimax Algorithm） に アルファ・ベータ枝刈り（Alpha-Beta Pruning） を組み合わせた高速な思考。

難易度（探索の深さ）を1〜10段階で調整可能。

モダンなGUI:

egui / eframe による軽量・高速な描画。

数値表示ではなく、石の数を視覚的に表現するカスタムペインターを実装。

型安全な設計:

Rustの所有権システムを活用し、AIの思考シミュレーション時も安全かつ効率的に盤面を複製。

🛠️ インストール・実行方法
Rustのインストール:
rustup.rs に従い、Rustツールチェーンをインストールしてください。

プロジェクトの準備:

Bash
git clone <your-repo-url>
cd rust_Mancala_game
実行:

Bash
cargo run --release
※ AIの深層探索をスムーズにするため、--release フラグを付けての実行を推奨します。

🎮 遊び方
プレイヤーの手番: 下段の6つの穴のうち、石が入っている場所をクリックして選択します。

AIの手番: あなたの手が終わると、設定された難易度に基づいてAIが自動的に最善手を指します。

難易度変更: 画面下のスライダーでAIのレベルを調整できます（Lv10は非常に強力です）。

ゲームの終了: どちらかの陣地の石がなくなると終了です。最終的に自分のストア（右側）に多くの石を集めた方の勝利となります。
